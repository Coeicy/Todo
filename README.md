# WeTodo 小程序

一个基于微信小程序的待办事项管理应用。

## 功能特性

- 任务管理
  - 创建、编辑、删除任务
  - 设置任务优先级
  - 添加任务备注、截止时间
  - 标记任务完成状态
  
- 日历视图
  - 农历/节假日显示
  - 任务日历视图
  - 日程安排

## 快速开始

1. 克隆项目
```bash
git clone https://github.com/your-username/wetodo.git
```

2. 安装依赖
```bash
npm install
```

3. 在微信开发者工具中导入项目

4. 开始开发

## 项目结构
